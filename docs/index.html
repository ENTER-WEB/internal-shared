How to Add a Manual

Claude Code にこれを貼るだけ。 ○○○ をマニュアルのテーマに置き換えてください。それ以外の操作は不要です。

Claude Code Prompt 
ENTER-WEB/internal-shared に「○○○のマニュアル」を追加して

これだけで全部終わります。 Claude が自動で HTML を生成し、PDF を作り、GitHub に push、Drive にアップロードまで完了させます。 手動の操作・確認・コピペは一切不要。

裏で実行される内容（クリックで詳細）
  1. ヒアリング：テーマ・元ネタ・コピペ対象を確認
  2. HTML生成：VAIZO Editorial 流儀＋コピーボタン＋隠しトグル付きで作成
  3. PDF生成：Playwright で A4・隠しトグル全展開・配布用に出力
  4. GitHub push：ENTER-WEB/internal-shared の docs/ と pdf/ に配置・commit・push
  5. Drive アップロード：ENTER WEB フォルダに同一PDFを配置
  6. 報告：GitHub Pages 公開URLと Drive PDF URL を返答
