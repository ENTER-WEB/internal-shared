# ENTER WEB 社内共通マニュアル

**公開URL**: https://enter-web.github.io/internal-shared/

---

## 使い方

Claude Code に **これだけ** 言えばOK：

```
ENTER-WEB/internal-shared に「○○○のマニュアル」を追加して
```

Claude が以下を全部やります：
- HTML マニュアル生成（VAIZO Editorial・コピペボタン・隠しトグル付き）
- PDF 生成
- repo に commit & push
- Drive の ENTER WEB フォルダに PDF アップロード
- GitHub Pages 公開URL を返答

詳細な作法は [`CLAUDE.md`](./CLAUDE.md) に記載済み。Claude が自動で読みます。
